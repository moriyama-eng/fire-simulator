# FIRE モンテカルロ・シミュレータ (Version 1.1.0)

本ツールは、ブラウザ単体で手軽に実行できる個人の資産形成・取崩しシミュレータです。
確率論的なアプローチ（幾何ブラウン運動とインフレ変動モデル）を用いて、運用資産の枯渇リスクや長期的な資産推移を可視化することを目的に制作しました。

## 特徴

- 統計的な価格変動モデルに基づくリスクシミュレーション
  幾何ブラウン運動（対数正規モデル）を採用し、設定した期待リターンとボラティリティに基づく市場価格の推移をシミュレーションします。
- インフレ変動モデル (AR-1モデル) の搭載
  単なる定率インフレだけでなく、統計データ（米国CPIなど）の特性を参考にしたAR-1（自己回帰）モデルを選択できます。これにより、物価上昇が長引くようなインフレリスク（スタグフレーション等）を考慮したシミュレーションが可能です。
- 動的な現金バッファ戦略
  運用資産が大きく下落した際（ドローダウン）、リスク資産の売却を避け、一時的に現金バッファを取崩し対象とする「現金クッション戦略」を組み込んでいます。総資産の状況に応じて、取崩しモードと回復時の補充モードが自動で切り替わります。
- 視覚的で分かりやすいUI
  ダークモードを基調とし、パーセンタイル（10%, 30%, 50%, 70%, 90%）を利用して資産分布をグラフ化（Chart.js）しています。各種設定項目はグループ化しており、初心者から上級者まで扱いやすいよう設計しています。

## 最新の更新 (v1.1.0)

- **デフォルト設定の最適化**
  米国株インデックス（S&P500）の長期実績に基づき、資産1億円・期待リターン10%・成功率90%超の安定したFIREモデルを初期値として採用しました。
- **画像保存 & X連携機能**
  シミュレーション結果を画像 (PNG) として保存したり、X（旧Twitter）へポストしたりする機能を追加しました。
- **UI/UXのブラッシュアップ**
  タイトル説明文の平易化や、モンテカルロ法の解説（ツールチップ）を追加。また、各説明文の表現をより自然で分かりやすく改善しました。
- **アクセス性の向上**
  フッターに GitHub リポジトリへのリンクを追加し、ソースコードへのアクセスを容易にしました。

## 開発の背景

私自身は機械設計を本業としており金融の専門家ではありませんが、自身の資産形成におけるリスクをより現実的に評価するために、複雑な計算をブラウザで直感的に行えるツールが必要と考え本シミュレータを開発しました。
同じようにFIREを目指す方々の参考になれば幸いです。

## 使い方

必要な環境構築はありません。リポジトリ内の `index.html` をブラウザで開くだけで動作します。
