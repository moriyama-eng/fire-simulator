# FIRE モンテカルロ・シミュレータ (v1.3.1)

本ツールは、ブラウザ単体で手軽に実行できる個人の資産形成・取崩しシミュレータです。
確率論的なアプローチ（幾何ブラウン運動とインフレ変動モデル）を用いて、運用資産の枯渇リスクや長期的な資産推移を可視化することを目的に制作しました。

## 特徴

- 統計的な価格変動モデルに基づくリスクシミュレーション
  幾何ブラウン運動（対数正規モデル）を採用し、設定した期待リターンとボラティリティに基づく市場価格の推移をシミュレーションします。
- テールリスク（最大下落・停滞期間）の可視化
  単なる資産推移だけでなく、最悪ケースのドローダウンや回復までの期間を、累積確率分布グラフ（CDF/CCDF）で直感的に把握可能です。特定のドローダウンを下回る確率や、停滞が長引くリスクを統計的に評価できます。
- インフレ変動モデル (AR-1モデル) の搭載
  単なる定率インフレだけでなく、統計データ（米国CPIなど）の特性を参考にしたAR-1（自己回帰）モデルを選択できます。これにより、物価上昇が長引くようなインフレリスク（スタグフレーション等）を考慮したシミュレーションが可能です。

## 主な更新履歴

### 更新内容 (v1.3.1)

- **ツールチップ表示UXの改善（モバイル対応強化）**
  スマートフォン等の狭い画面でツールチップが画面外にはみ出し、意図しない横スクロールが発生する問題を修正。要素の配置場所に応じて表示位置を調整（左寄せ/中央寄せ）するように改善し、視認性と操作性を向上させました。
- **UI全体のバージョン表記統一**
  アプリ内の各所（タイトル、フッター、保存画像クレジット）のバージョン表記を「v1.3.1」形式に統一。

### 更新内容 (v1.3.0)

- **リスク分析グラフの高度化 (CDF/CCDF採用)**  
  「最大ドローダウン」と「最長停滞期間」のグラフを累積確率分布（CDF/CCDF）に刷新。特定のラインを「下回る/上回る」確率を一目で把握できる、より実戦的なリスク評価ツールへ進化。
- **「資産枯渇（破綻）」の明示的表現**  
  最大ドローダウンが100%に達するシナリオ（資産枯渇）に対し、グラフ上で「総資産が枯渇する確率」として直感的な表示をサポート。
- **描画パフォーマンスの向上**  
  シミュレーションパスが大量（10万パス等）であっても、代表点を抽出する「間引き描画（Decimation）」を導入。精緻な確率分布を維持しつつ、超軽量・高速なブラウザ描画を実現。
- **出力画像デザインの洗練**  
  出力画像のレイアウトを刷新し、タイトルと実行日時の視認性を向上。

### 更新内容 (v1.2.0)

- **リスク分析グラフの追加**  
  「最大ドローダウン」と「最長停滞期間」の確率密度分布（PDF）を実装。ワーストケースの発生頻度を統計的に把握可能に。
- **計算待ち時間の快適化**  
  シミュレーション実行中、ボタン自体がプログレスバー化。完了時の集計待ち時間を改善する手法（Yield描画）を導入し、体感速度を改善。
- **細かなUI/UXの改善**  
  全ての主要グラフに詳細な解説ツールチップ（ℹ️マーク）を設置。グラフ軸の単位（年表記）や目盛りの統一感を向上。また、計算完了時の結果エリアへの自動スクロール機能を搭載。
- **設定値入力の補助機能追加**  
  極端な数値入力に対するリアルタイムの補正機能と視覚的な通知を実装。

## 開発の背景

私自身は機械設計を本業としており金融の専門家ではありませんが、自身の資産形成におけるリスクをより現実的に評価するために、複雑な計算をブラウザで直感的に行えるツールが必要と考え、本シミュレータを開発しました。
同じようにFIREを目指す方々の参考になれば幸いです。

## 使い方

必要な環境構築はありません。リポジトリ内の `index.html` をブラウザで開くだけで動作します。
