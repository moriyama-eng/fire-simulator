# リリース前スモークテストマニュアル

新バージョンリリース時には、以下のチェックリストを用いて手動確認を行ってください。

## シミュレーションタブ
- [ ] `npm test` が全パスする
- [ ] 「シミュレーション実行」ボタンで結果が表示される
- [ ] サマリーカードに成功率・最終総資産中央値・目標資産維持確率が表示される
- [ ] 総資産推移グラフが描画される
- [ ] ダウンサイドフォーカストグルが動作する（50%以下のみ表示）
- [ ] 「画像を保存」が成功する
- [ ] 「Xにポスト」で正しいURLが生成される
- [ ] 「同条件の別タブを開く」が動作する
- [ ] 「解析結果のURLリンクをコピー」が動作する

## 分析タブ
- [ ] 分析タブを開くと基準シナリオカードが表示される
- [ ] 因子カードのクリックで選択/解除が動作する
- [ ] 選択因子数・シナリオ数が正しく更新される
- [ ] 「分析を実行」で比較表が表示される
- [ ] ターゲットテーブルが正しく表示される
- [ ] 評価指標切替でテーブルとラベルが更新される
- [ ] 「ZIP出力 (ベータ)」が成功し、ファイルがダウンロードされる
- [ ] ZIPに summary.csv, comparison_summary.csv, metadata/*.json, manifest.json が含まれる
- [ ] 因子全解除で「因子を選択してください」が表示される
- [ ] 「シミュレーションタブで条件を編集」ボタンでタブ切替が動作する

## クロスブラウザ確認
- [ ] Chrome最新版で正常動作
- [ ] モバイル表示（レスポンシブ）が崩れていない

## 英語モードの確認

- [ ] 言語切替ボタン「English」をクリックすると UI が英語に切り替わる
- [ ] 英語モード切り替え後、UI 内に日本語のテキストが残っていない（「日本語」ボタン自体を除く）
- [ ] シミュレーションタブの通貨表示が正しい単位（M, K）で表示される
  - 例: 初期リスク資産 1.0億円 → `$1.0 M`
  - 例: 初期現金バッファ 1,000万円 → `$100 K`
- [ ] サマリーカードの「最終総資産 中央値」が `$X.X M` など正しく表示される
- [ ] 分析タブの因子「初期リスク資産」「初期現金バッファ」の基準値が正しい USD 単位で表示される
- [ ] 分析タブのターゲットテーブルで、長い因子名（例: "Initial cash buffer"）がセル内で折り返されず、はみ出す場合は横スクロールまたは省略記号で対応している
- [ ] 英語モードでステッパーボタン操作時に手動入力した小数点以下の値が消失しない
- [ ] 総資産グラフのツールチップに `$X.X M` など正しい通貨単位が表示される
- [ ] 「画像を保存」で生成される PNG 内の通貨表示が英語モードに準拠している
- [ ] X ポスト、URL コピー、別タブ開く の各共有機能が英語モードでも正常に動作する
