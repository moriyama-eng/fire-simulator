# リリース前スモークテストマニュアル

新バージョンリリース時には、以下のチェックリストを用いて手動確認を行ってください。

## シミュレーションタブ
- [ ] `npm test` が全パスする
- [ ] 「シミュレーション実行」ボタンで結果が表示される
- [ ] サマリーカードに成功率・最終総資産中央値・目標資産維持確率が表示される
- [ ] 総資産推移グラフが描画される
- [ ] ダウンサイドフォーカストグルが動作する（50%以下のみ表示）
- [ ] 「画像を保存」が成功する
- [ ] 「Xにポスト」で正しいURLが生成される
- [ ] 「同条件の別タブを開く」が動作する
- [ ] 「解析結果のURLリンクをコピー」が動作する

## 分析タブ
- [ ] 分析タブを開くと基準シナリオカードが表示される
- [ ] 因子カードのクリックで選択/解除が動作する
- [ ] 選択因子数・シナリオ数が正しく更新される
- [ ] 「分析を実行」で比較表が表示される
- [ ] ターゲットテーブルが正しく表示される
- [ ] 評価指標切替でテーブルとラベルが更新される
- [ ] 「ZIP出力 (ベータ)」が成功し、ファイルがダウンロードされる
- [ ] ZIPに summary.csv, comparison_summary.csv, metadata/*.json, manifest.json が含まれる
- [ ] 因子全解除で「因子を選択してください」が表示される
- [ ] 「シミュレーションタブで条件を編集」ボタンでタブ切替が動作する

## 比較タブ
- [ ] 比較タブを開くと1つのシナリオが表示される（日本語・英語両方で）
- [ ] シナリオ追加・削除・複製・上書きが正しく動作する
- [ ] 削除確認ダイアログで「キャンセル」を選択した場合、シナリオが削除されない
- [ ] 左右移動ボタンで列の順序を変更できる
- [ ] ドラッグまたはメニューボタンで並び替えができる旨のヒントが画面上部に常時テキスト表示される
- [ ] 共通シード値・共通パス数を変更できる
- [ ] 入力値の範囲外クランプ時に赤枠アニメーションが表示される
- [ ] 数値入力後、Tabキーで次のフィールドにフォーカスが移動する
- [ ] 「すべて実行」で全シナリオが逐次実行される
- [ ] 実行中は全ての操作ボタンが無効化される
- [ ] 実行中に共通設定（シード・パス数）が変更できないことを確認する
- [ ] 実行中に進捗表示が更新される（各シナリオ完了ごとにテーブルが再描画され、ボタン文言が更新される）
- [ ] 日本語モードで初期リスク資産の右側に「億円」と表示される
- [ ] 日本語モードで初期現金バッファの右側に「万円」と表示される
- [ ] 日本語モードで補充ペースの右側に「倍」と表示される
- [ ] 英語モードで通貨単位が「M」「K」など正しく表示される
- [ ] 英語モードで補充ペースの右側に「x」と表示される
- [ ] 英語モードで数値入力後、日本語モードに戻しても値が正しい
- [ ] 英語モードで増減ステップ・最小・最大値も正しく変換される
- [ ] 英語モードで targetAssetRatio を編集後も値が維持される
- [ ] スマホ表示で横スクロール・1列目固定が機能する
- [ ] テーブル再描画後にスクロール位置が維持されることを確認する
- [ ] ツールチップ（ℹ️アイコン）がキーボードフォーカス可能
- [ ] CB OFFのシナリオではCB関連パラメータが編集不可（グレーアウト）
- [ ] GR OFFのシナリオではGR関連パラメータが編集不可（グレーアウト）
- [ ] シナリオ名編集後、即座に画面上の名前が更新される
- [ ] シナリオ名編集中に言語切り替えなどの外部イベントが発生しても、編集中の内容が失われないまたは適切にフォーカスが復元される
- [ ] シナリオ名編集中に「すべて実行」ボタンを押すと、編集中の内容が確定された上で実行が開始される
- [ ] セレクトボックス（変動モデル、t分布自由度、インフレ変動モデル）を変更しても、シミュレーションがエラーなく実行できること（コンソールに NaN エラーが出ないこと）
- [ ] タブ切り替え（シミュレーション ⇔ 分析 ⇔ 比較）を繰り返しても、タブコンテンツが重ならず、常に1つのタブだけが表示されること
- [ ] 分析タブの同期確認: シミュレーションタブで実行 → 分析タブで基準シナリオが一致すること
- [ ] 変数宣言の確認: ブラウザの開発者コンソールで `ReferenceError` が発生していないこと（特に `comparisonTabBtn` 未定義エラーが出ていないこと）
- [ ] 言語切り替えの二重実行防止: 言語切替ボタンを複数回クリックしても、再描画が重複して実行されないことを確認する
- [ ] テーブル閉じタグの最終確認: 開発者ツールで `</tbody><tr></div>` の順序が正しいこと
- [ ] 循環インポートの確認: ブラウザコンソールで `TypeError: getParams is not a function` などのエラーが出ていないことを確認する

## クロスブラウザ確認
- [ ] Chrome最新版で正常動作
- [ ] モバイル表示（レスポンシブ）が崩れていない

## 英語モードの確認

- [ ] 言語切替ボタン「English」をクリックすると UI が英語に切り替わる
- [ ] 英語モード切り替え後、UI 内に日本語のテキストが残っていない（「日本語」ボタン自体を除く）
- [ ] シミュレーションタブの通貨表示が正しい単位（M, K）で表示される
  - 例: 初期リスク資産 1.0億円 → `$1.0 M`
  - 例: 初期現金バッファ 1,000万円 → `$100 K`
- [ ] サマリーカードの「最終総資産 中央値」が `$X.X M` など正しく表示される
- [ ] 分析タブの因子「初期リスク資産」「初期現金バッファ」の基準値が正しい USD 単位で表示される
- [ ] 分析タブのターゲットテーブルで、長い因子名（例: "Initial cash buffer"）がセル内で折り返されず、はみ出す場合は横スクロールまたは省略記号で対応している
- [ ] 英語モードでステッパーボタン操作時に手動入力した小数点以下の値が消失しない
- [ ] 総資産グラフのツールチップに `$X.X M` など正しい通貨単位が表示される
- [ ] 「画像を保存」で生成される PNG 内の通貨表示が英語モードに準拠している
- [ ] X ポスト、URL コピー、別タブ開く の各共有機能が英語モードでも正常に動作する
